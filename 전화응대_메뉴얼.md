# 오늘은 봄날 전화응대 핸드북

대구 여성기업 오늘은 봄날의 공공예술 인터랙티브 콘텐츠 문의를 받을 때 사용하는 응대 매뉴얼입니다.

## 핵심 원칙

- 바로 가격을 확정하지 말고, 장비와 현장 조건을 먼저 확인합니다.
- 예산을 먼저 물어보고, 예산 안에서 가능한 범위를 안내합니다.
- LED, 빔프로젝터, 야외 설치는 현장 사진이나 실측이 필요하다고 안내합니다.
- 모르는 기술 질문은 즉답하지 말고 확인 후 다시 연락합니다.

## 1. 첫 인사

안녕하세요. 오늘은 봄날입니다.

미디어아트나 공공예술 콘텐츠 설치 문의 주신 걸까요?

어떤 공간에 설치를 생각하고 계신지 간단히 여쭤봐도 될까요?

## 2. 기본 정보 확인

| 확인 항목 | 질문 예시 |
| --- | --- |
| 기관/회사명 | 문의 주신 기관명이나 회사명을 알 수 있을까요? |
| 담당자 | 담당자 성함과 연락처를 남겨주실 수 있을까요? |
| 설치 지역 | 설치 장소는 어느 지역인가요? |
| 공간 종류 | 로비, 야외 광장, 전시장, 학교, 공공기관 중 어떤 공간인가요? |
| 설치 목적 | 전시, 홍보, 축제, 야간경관, 체험 콘텐츠 중 어떤 목적일까요? |
| 희망 일정 | 언제까지 설치나 운영을 원하시나요? |

## 3. 장비 유형별 질문

### LED 화면

- LED 화면이 이미 설치되어 있나요?
- 화면 가로, 세로 크기를 알고 계신가요?
- 해상도나 픽셀피치를 알고 계신가요?
- 실내 LED인가요, 야외 LED인가요?
- 송출 PC가 이미 있나요?
- 운영 시간이 하루 몇 시간 정도인가요?

안내 멘트:

LED는 화면 크기와 해상도에 따라 제작 방식과 견적이 많이 달라집니다. 가능하면 현장 사진과 화면 규격을 받아본 뒤 견적서를 드리는 게 정확합니다.

### 빔프로젝터

- 프로젝터가 이미 있나요, 새로 설치해야 하나요?
- 투사할 벽면이나 건물 외벽 크기는 어느 정도인가요?
- 실내인가요, 야외인가요?
- 밤에만 운영하나요, 낮에도 보여야 하나요?
- 프로젝터를 설치할 위치와 전원 위치가 확보되어 있나요?
- 프로젝션 맵핑이 필요한 형태인가요?

안내 멘트:

빔프로젝터는 밝기, 투사 거리, 주변 조도에 따라 장비가 달라집니다. 특히 야외는 현장 실측이 거의 필요합니다.

### 키오스크 / TV / 미디어월

- 화면 장비가 이미 있나요?
- 터치가 필요한가요?
- 인터넷 연결이 가능한가요?
- 미니 PC 또는 기존 송출 PC를 사용할 수 있는 환경인가요?
- 부팅하면 자동으로 작품이 실행되면 될까요?

안내 멘트:

키오스크나 TV형 콘텐츠는 비교적 빠르게 설치할 수 있습니다. 날씨, 미세먼지, 온도 같은 실시간 데이터를 연결할 수도 있습니다.

## 4. 실측 필요 여부

- 현장 사진을 보내주실 수 있을까요?
- 화면이 설치될 위치의 가로, 세로 크기를 알 수 있을까요?
- 관람자가 보는 거리는 어느 정도인가요?
- 전원과 인터넷 위치는 가까운가요?
- 설치물이 실내인지 야외인지 확인 가능할까요?

안내 멘트:

정확한 견적을 위해서는 사진, 대략적인 사이즈, 장비 보유 여부가 필요합니다. 야외 LED나 빔프로젝션은 현장 실측 후 견적이 가장 정확합니다.

## 5. 예산 확인

예산 범위에 맞춰 콘텐츠 범위와 장비 구성을 조정해드릴 수 있습니다.

혹시 이번 사업이나 프로젝트의 예상 예산 범위가 있을까요?

대략적인 범위만 알려주셔도 됩니다.

예산 질문 예시:

- 500만 원 정도의 키오스크형 콘텐츠를 생각하시나요?
- 1,000만 원 정도의 터치/센서형 콘텐츠를 생각하시나요?
- 1,000만 원 이상 공공 프로젝트로 검토 중이신가요?
- 장비비와 콘텐츠 제작비가 포함된 예산인가요?
- 장비는 이미 있고 콘텐츠만 필요한 상황인가요?

## 6. 견적 기준

정확한 금액은 현장 조건에 따라 달라진다고 꼭 말합니다.

| 유형 | 기준 금액 | 설명 |
| --- | ---: | --- |
| 웹/키오스크형 미디어아트 | 500만 원 | 기존 키오스크나 화면 장비가 있고 콘텐츠 중심일 때 |
| 실시간 데이터 연동 콘텐츠 | 500만 원 | 날씨, 미세먼지, 온도 등 API 연동 포함 |
| 터치/센서 인터랙티브 콘텐츠 | 1,000만 원부터 | 터치스크린, 센서, 관람자 반응 포함 |
| 실내 미디어월 콘텐츠 | 1,000만 원부터 | 공간 맞춤 화면 제작 및 설치 세팅 |
| 야외 미디어파사드 | 1,500만 원부터 | 현장 실측, 장비, 밝기, 송출 테스트 필요 |
| 프로젝션 맵핑 | 2,000만 원부터 | 투사 거리, 맵핑, 야간 운영 조건에 따라 변동 |

안내 멘트:

정확한 견적은 공간 크기, 장비 보유 여부, 인터랙션 방식, 운영 기간에 따라 달라집니다. 먼저 기본 정보를 받아보고 견적서를 정리해서 보내드리겠습니다.

## 7. 견적서 발송 안내

말씀해주신 내용을 바탕으로 내부 검토 후 견적서를 보내드리겠습니다.

견적서 받으실 이메일 주소를 알려주실 수 있을까요?

가능하시면 현장 사진, 화면 크기, 장비 정보도 함께 보내주세요.

## 8. 미팅 제안 기준

다음 중 하나라도 해당하면 미팅을 제안합니다.

- 야외 설치
- LED 또는 빔프로젝터 사용
- 예산 1,000만 원 이상
- 공공기관/지자체 사업
- 현장 실측 필요
- 인터랙션 방식이 복잡함

멘트:

이 건은 현장 조건을 같이 봐야 정확할 것 같습니다.

가능하시면 30분 정도 온라인 미팅을 잡고, 공간 사진이나 도면을 보면서 방향을 정리해드리겠습니다.

## 9. 전화 마무리

오늘 말씀해주신 내용 정리해서 확인 후 연락드리겠습니다.

현장 사진과 화면 크기, 장비 정보, 예산 범위를 보내주시면 더 정확하게 견적드릴 수 있습니다.

문의 주셔서 감사합니다.
