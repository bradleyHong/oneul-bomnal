Daegu Women-led Public Art Studio

대구 여성기업 오늘은 봄날은 공공예술 인터랙티브 콘텐츠 전문회사입니다.

지자체, 공공기관, 문화재단, 축제, 학교, 도서관, 도시재생 공간을 위한 미디어아트를 기획하고 제작합니다. 공공사업 랜딩페이지, 설문조사, 참여 신청, 결과 아카이브 페이지까지 함께 만들어 시민이 보고 참여하고 남길 수 있는 디지털 접점을 설계합니다. 미디어아트 화면 안에 QR을 심어 랜딩페이지, 설문, 이벤트, 사은품 수령 페이지로 연결할 수 있고, 날씨·미세먼지·수위·교통·행사 데이터 같은 공공 API 조건에 따라 서로 다른 결과물이 나오도록 제작할 수 있습니다.

작품별 공공 데이터 연결 방식
작품 연결 데이터 시민에게 보여주는 판단 적용 공간 설치 장비
오늘은 봄날 대구 날씨, 바람, 구름, 강수 오늘의 하늘과 현재 시각을 시침, 초침, 꽃잎, 빛의 움직임으로 표현 공공청사 로비, 미디어월, 야외 LED 현장 송출용 미니 PC 권장
빛의 산책로 대구 미세먼지 API, 초미세먼지 지수 아이들이 공원에서 야외활동을 해도 되는지 색과 신호로 안내 공원, 학교 앞, 어린이 체험 공간 미니 PC 권장, 네트워크 연결 필요
숨 쉬는 정원 오늘 온도, 체감온도, 시간대별 변화 따뜻함과 추위를 꽃의 크기, 색온도, 호흡 속도로 시각화 도서관, 병원, 복지관, 웰니스 공간 4K 장시간 운영은 미니 PC 권장
도시의 꽃비 강수 확률, 강수량, 기상 예보 오늘 비가 올지 안 올지를 꽃비의 밀도와 하늘 변화로 표현 축제 미디어파사드, 광장, 관광 안내 공간 미니 PC 송출, 프로젝션/LED 현장 테스트 권장

Company Profile

오늘은 봄날 회사소개서

Daegu Women-owned Public Art Company

공공데이터, 미디어아트, 시민 참여 페이지를 하나의 프로젝트로 설계합니다.

오늘은 봄날은 대구 여성기업으로, 공공기관과 지자체의 사업 목적에 맞춰 데이터 기반 미디어아트, LED 사이니지 콘텐츠, 랜딩페이지, 설문·신청 시스템을 함께 제작합니다.

01

데이터 기반 미디어아트

날씨, 미세먼지, 교통, 수위, 행사 참여 데이터 등 다양한 API 조건에 따라 화면 결과가 달라지는 콘텐츠를 제작합니다.

02

LED·사이니지 송출

공공청사, 도서관, 축제장, 야외 LED, 실내 미디어월에 맞춰 화면 비율과 운영 환경을 설계합니다.

03

QR 참여 연결

작품 화면 안의 QR을 통해 랜딩페이지, 설문, 이벤트, 사은품 수령, 신청접수 페이지로 연결합니다.

04

B2G 프로젝트 문서화

기획 의도, 장비 조건, 운영 방식, 결과 데이터 정리까지 공공사업 보고에 필요한 흐름을 준비합니다.

Artist

홍석범 대구경북지역 미디어아티스트

대구경북지역을 기반으로 빛, 꽃잎, 바람, 계절감을 디지털 풍경으로 번역합니다. 장소의 표면과 관람 거리에 맞춰 작품의 밀도와 리듬을 조율합니다.

Weather Responsive

공공 데이터와 장소 반응

날씨, 시간, 관람자 움직임, 지역 데이터를 작품의 색과 흐름으로 바꿉니다. 프로젝트별로 필요한 데이터 연동을 설계합니다.

Interactive

B2G 맞춤형 제작

공공기관의 예산, 일정, 공간, 장비 조건에 맞춰 콘텐츠 범위와 설치 방식을 현실적으로 조정합니다.

Project Flow

제안서 단계부터 현장 운영까지 공공 프로젝트 흐름에 맞춰 움직입니다.

  1. 상담

    사업 목적, 대상 공간, 예산 범위, 공고 일정, 운영 기간을 확인합니다.

  2. 제안

    공공예술 기획 의도, 콘텐츠 구성, 장비 조건, 운영 방식을 제안서 형태로 정리합니다.

  3. 제작

    작품 그래픽, 인터랙션, 데이터 연동, 현장 화면 비율을 맞춰 제작합니다.

  4. 운영

    설치 테스트, 송출 안정화, 운영 매뉴얼, 행사 기간 대응까지 준비합니다.
